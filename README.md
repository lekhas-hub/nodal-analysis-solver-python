# Nodal Analysis Solver (Python)

## 📌 Description
This project calculates node voltage in a circuit using nodal analysis (KCL).

## 🚀 Features
- Accepts resistor values and current source
- Computes node voltage
- Based on circuit analysis principles

## 🧠 Concepts Used
- Network Analysis
- Kirchhoff’s Current Law (KCL)
- Conductance method

## ▶️ How to Run
Run:
python main.py
